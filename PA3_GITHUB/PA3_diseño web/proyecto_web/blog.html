Sabores que reconectan.



En un mundo que corre, una taza de té puede ser una pausa. No solo se trata de beber algo caliente, sino de regalarte un momento.

Preparar té es un acto simple, pero poderoso: hervir agua, elegir tu mezcla, oler las hierbas antes de sumergirlas... ese instante es tuyo.

En Light of Leaves, creemos que cada infusión puede acompañar una emoción: calmar, despertar, inspirar o simplemente abrazarte. Porque el té no solo se toma, también se siente. Respira, bebe, reconecta.

El té es más que una bebida: es bienestar en cada sorbo. Rico en antioxidantes, ayuda a fortalecer el sistema inmunológico, mejora la digestión y reduce el estrés. Cada variedad aporta algo único: el té verde energiza, el negro concentra, el rooibos relaja, y las infusiones herbales reconectan cuerpo y mente.

Una pausa con té es una caricia para el día.
